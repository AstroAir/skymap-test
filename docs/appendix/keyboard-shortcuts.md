# 键盘快捷键

本文档列出 SkyMap Test 中可用的所有键盘快捷键。

## 全局快捷键

### 文件操作

| 快捷键 | 功能 |
|--------|------|
| Ctrl+O | 打开文件 |
| Ctrl+S | 保存当前配置 |
| Ctrl+P | 打印预览 |
| Ctrl+Q | 退出应用 |

### 编辑操作

| 快捷键 | 功能 |
|--------|------|
| Ctrl+F | 搜索天体 |
| Ctrl+C | 复制 |
| Ctrl+V | 粘贴 |
| Ctrl+Z | 撤销 |
| Ctrl+Y | 重做 |

## 星图控制

### 视角控制

| 快捷键 | 功能 |
|--------|------|
| ↑ | 向上旋转视角 |
| ↓ | 向下旋转视角 |
| ← | 向左旋转视角 |
| → | 向右旋转视角 |
| Home | 跳转到天顶 |
| End | 跳转到天底 |

### 缩放控制

| 快捷键 | 功能 |
|--------|------|
| + 或 = | 放大 |
| - 或 _ | 缩小 |
| * | 最大放大 |
| / | 最小缩小 |
| 0 | 重置缩放 |

### 时间控制

| 快捷键 | 功能 |
|--------|------|
| T | 跳转到当前时间 |
| Space | 暂停/继续时间 |
| , | 时间倒流 |
| . | 时间加速 |
| / | 恢复正常速度 |

## 视图切换

| 快捷键 | 功能 |
|--------|------|
| F1 | 显示/隐藏侧边栏 |
| F2 | 显示/隐藏信息面板 |
| F3 | 显示/隐藏工具栏 |
| F4 | 显示/隐藏状态栏 |
| F5 | 刷新星图 |
| F11 | 全屏模式 |

| 快捷键 | 功能 |

| B | 切换背景颜色 |

## 搜索快捷键

### 在搜索框中

| 快捷键 | 功能 |
|--------|------|
| Esc | 关闭搜索 |
| Enter | 搜索 |
| ↓ | 下一个结果 |
| ↑ | 上一个结果 |
| Tab | 自动完成 |

## 数字快捷键

| 快捷键 | 功能 |
|--------|------|
| 1-9 | 快速切换到对应预设视角 |
| Ctrl+1-9 | 快速切换到对应配置 |

## 自定义快捷键

您可以在设置中自定义快捷键：

1. 打开设置（Ctrl+,）
2. 选择"键盘"选项卡
3. 点击要修改的快捷键
4. 按下新的组合键
5. 保存设置

## 快捷键冲突

如果快捷键与其他应用冲突：

1. 打开快捷键设置
2. 搜索冲突的快捷键
3. 修改或禁用该快捷键
4. 保存设置

## 游戏手柄支持

SkyMap Test 支持游戏手柄：

- 左摇杆：旋转视角
- 右摇杆：缩放
- 方向键：时间控制
- A按钮：确认
- B按钮：取消

## 触摸手势（触摸屏设备）

- 单指拖动：旋转视角
- 双指捏合：缩放
- 三指上滑：显示侧边栏
- 三指下滑：隐藏侧边栏
