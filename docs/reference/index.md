## 天文学基础

### 坐标系统

理解天文学坐标系统对于使用 SkyMap Test 非常重要：

- **地平坐标系**：以观测者为基准
- **赤道坐标系**：以天赤道为基准
- **银河坐标系**：以银河平面为基准
- **黄道坐标系**：以黄道为基准

### 时间系统

天文学使用多种时间系统：

- **世界时 (UT)**：基于地球自转
- **协调世界时 (UTC)**：使用跳秒的UT
- **恒星时**：以恒星为基准
- **儒略日**：连续的日计数

### 天体运动

- **周日运动**：地球自转导致的运动
- **周年运动**：地球公转导致的运动
- **行星运动**：行星相对于恒星的复杂运动

## 性能指标

### 星等系统

理解星等系统：

- **视星等**：地球上的观测亮度
- **绝对星等**：标准距离的亮度
- **摄影星等**：特定滤镜的测量

## 观测条件

### 大气影响

地球大气对观测的影响：

- **大气消光**：吸收和散射星光
- **大气折射**：改变天体视位置
- **视宁度**：影响图像清晰度
- **光污染**：降低可见极限星等

### 最佳观测条件

- **黑暗天空**：远离城市光害
- **晴朗天气**：无云或少云
- **低湿度**：减少大气散射
- **稳定大气**：良好视宁度

## 常用计算

### 角度计算

- 1度 = 60角分
- 1角分 = 60角秒
- 1度 = 3600角秒

### 时间计算

- 1天 = 24小时
- 1小时 = 60分钟
- 1分钟 = 60秒

### 距离单位

- **光年**：光一年走的距离（约9.46万亿公里）
- **秒差距**：约3.26光年
- **天文单位**：日地平均距离（约1.5亿公里）
