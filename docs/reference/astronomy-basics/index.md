## 基本概念

### 天球

天球是一个假想的无限大球体，观测者位于球心，所有天体都投影在球面上。

### 天极和天赤道

- **天极**：地球自转轴延长线与天球的交点
- **天赤道**：地球赤道平面与天球的交圆

### 黄道

黄道是太阳在天球上的视运动轨迹，与天赤道交角约 23.44°。

### 星等

星等是描述天体亮度的单位：

| 星等 | 可见性 | 示例天体 |
|-----|-------|---------|
| -27 | - | 太阳 |
| -13 | - | 满月 |
| -4 | 最亮 | 金星（最亮时） |
| 0 | 很亮 | 织女星 |
| 2 | 明显 | 北极星 |
| 6 | 肉眼极限 | 暗星 |
| 10 | 双筒 | 需要光学设备 |

### 角度单位

- **度 (°)**：圆周的 1/360
- **角分 (')**：1° = 60'
- **角秒 (")**：1' = 60"

常见参考：
- 满月直径：约 0.5°（30'）
- 木星直径：约 50"（最大时）
- 恒星视直径：< 0.1"

## 时间系统

### 世界时 (UT)

基于地球自转的时间系统，UT1 是主要形式。

### 协调世界时 (UTC)

基于原子时的国际标准时间，与 UT1 保持在 0.9 秒以内。

### 恒星时

基于恒星背景的时间系统，一个恒星日约 23 小时 56 分 4 秒。

### 儒略日 (JD)

从公元前 4713 年 1 月 1 日中午开始的连续计日系统。

## 天文现象

### 曙暮光

| 类型 | 太阳高度 | 特征 |
|-----|---------|------|
| 民用曙暮光 | 0° ~ -6° | 可进行户外活动 |
| 航海曙暮光 | -6° ~ -12° | 地平线可见 |
| 天文曙暮光 | -12° ~ -18° | 天空接近全黑 |

### 月相

月相周期约 29.5 天（朔望月），主要相位：

- 新月（朔）
- 上弦月
- 满月（望）
- 下弦月
