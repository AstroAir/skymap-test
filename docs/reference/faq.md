# 常见问题

本文档收集了用户常见的疑问和解答。

## 安装和设置

### Q: 应用无法启动怎么办？

**A**: 尝试以下步骤：

1. 确认系统满足最低要求
2. 重新安装应用
3. 检查防火墙设置
4. 查看错误日志

Windows: `%APPDATA%\SkyMapTest\logs\`
macOS: `~/Library/Logs/SkyMapTest/`
Linux: `~/.local/share/SkyMapTest/logs/`

### Q: 如何卸载应用？

**A**:
- Windows: 控制面板 > 程序和功能
- macOS: 删除 Applications 文件夹中的应用
- Linux: 删除 AppImage 文件

## 使用问题

### Q: 星图显示不正确？

**A**: 检查以下几点：

1. 确认观测位置设置正确
2. 确认时间设置准确
3. 检查显示设置中的星等限制
4. 尝试重置视图设置

### Q: 找不到某个天体？

**A**:
1. 确认天体名称正确
2. 尝试使用不同的编号系统
3. 检查星等限制设置
4. 确认该天体在当前星表数据中

### Q: 时间控制不工作？

**A**:
1. 确认未处于实时模式
2. 检查时间流逝速度设置
3. 尝试暂停后重新启动

## 性能问题

### Q: 应用运行缓慢？

**A**: 尝试以下优化：

1. 降低星等限制
2. 减少显示的图层
3. 关闭不需要的辅助线
4. 使用性能模式渲染
5. 更新到最新版本

### Q: 内存占用过高？

**A**:
1. 清理缓存数据
2. 减小缓存区域
3. 重启应用
4. 检查内存泄漏（如果持续增长）

## 数据和缓存

### Q: 如何下载数据用于离线使用？

**A**:
1. 打开缓存管理
2. 选择要下载的区域
3. 点击下载按钮
4. 等待下载完成

### Q: 缓存数据占用空间太大？

**A**:
1. 打开缓存管理
2. 查看缓存统计
3. 删除不需要的缓存
4. 调整缓存大小限制

### Q: 数据更新失败？

**A**:
1. 检查网络连接
2. 确认服务器状态
3. 尝试手动更新
4. 清理缓存后重试

## 设备管理

### Q: 如何添加望远镜？

**A**:
1. 打开设备管理器
2. 点击"添加望远镜"
3. 填写望远镜参数
4. 保存配置

### Q: 视野计算不准确？

**A**:
1. 检查设备参数是否正确
2. 确认望远镜和相机配置匹配
3. 验证焦距和传感器尺寸
4. 尝试重新计算

## 观测规划

### Q: 可见性预测不准确？

**A**:
1. 确认位置和时间设置准确
2. 检查地平线遮挡设置
3. 更新大气折射参数
4. 考虑当地天气条件

### Q: 如何导出观测列表？

**A**:
1. 打开目标列表
2. 点击"导出"按钮
3. 选择导出格式
4. 保存文件

## 安全问题

### Q: 应用数据是否安全？

**A**: SkyMap 采用多层安全防护：

1. **数据存储** - 所有数据以 JSON 格式存储在应用专用目录中
2. **路径沙箱** - 文件操作限制在应用数据目录内
3. **输入验证** - 所有用户输入都经过大小和格式验证
4. **速率限制** - 防止 API 滥用和资源耗尽

### Q: 网络请求是否安全？

**A**: 是的，SkyMap 实现了以下保护：

1. **HTTPS 强制** - 所有网络请求默认使用 HTTPS
2. **URL 验证** - 阻止访问私有 IP 和内部网络
3. **SSRF 防护** - 阻止服务端请求伪造攻击

### Q: 如何清除应用数据？

**A**:
1. 打开设置 → 高级
2. 点击"重置应用数据"
3. 确认操作

或手动删除数据目录：
- Windows: `%APPDATA%\SkyMapTest\`
- macOS: `~/Library/Application Support/SkyMapTest/`
- Linux: `~/.local/share/SkyMapTest/`

### Q: 应用会收集用户数据吗？

**A**: 不会。SkyMap 是本地应用，所有数据存储在本地，不会向任何服务器发送用户数据。
