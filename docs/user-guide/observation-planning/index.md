# 观测规划

SkyMap Test 提供强大的观测规划功能，帮助您充分利用每一个晴朗的夜晚。

## 功能概览

### 智能推荐

- **今晚推荐** - 根据当前条件推荐最佳观测目标
- **最佳时间** - 计算每个目标的最佳观测时段
- **月相考虑** - 自动避开月光干扰

### 目标管理

- **目标列表** - 创建和管理观测目标清单
- **优先级排序** - 根据可见性和条件自动排序
- **状态追踪** - 记录观测进度

### 时间规划

- **可见性图表** - 显示目标高度随时间变化
- **曙暮光时间** - 精确计算天文昏影时间
- **会话规划** - 优化多目标观测顺序

## 核心组件

### 今晚推荐面板

系统自动分析当前条件，推荐最适合观测的目标：

- 根据位置计算目标可见性
- 考虑月相和天气条件
- 按推荐分数排序显示
- 一键添加到目标列表

### 天空图集面板

浏览和搜索深空天体目录：

- Messier、NGC、IC 目录
- 高级筛选（类型、亮度、大小）
- 模糊搜索支持
- 详细天体信息

### 高度图表

可视化目标在夜间的高度变化：

- 实时高度曲线
- 子午线标记
- 薄暮时段显示
- 多目标对比

### 曝光计算器

天文摄影曝光参数计算：

- 信噪比估算
- 最佳曝光时间
- 叠加帧数建议
- 设备参数配置

## 使用流程

### 1. 查看今晚条件

打开应用后，首先查看：

- 日落/日出时间
- 天文昏影时段
- 当前月相
- 天气预报（如果可用）

### 2. 选择观测目标

通过以下方式选择目标：

- 查看系统推荐
- 在天空图集中搜索
- 从历史记录中选择
- 手动输入坐标

### 3. 规划观测顺序

使用规划工具优化顺序：

- 按高度优先
- 按子午线时间
- 避免频繁换向
- 考虑设备限制

### 4. 执行观测

观测过程中：

- 实时查看目标状态
- 记录观测笔记
- 标记完成状态
- 添加新发现
