# 目标列表

本文档介绍如何使用 SkyMap Test 的目标列表功能管理观测目标。

## 目标列表概览

### 什么是目标列表

目标列表是您计划观测的天体集合：

- 组织待观测目标
- 跟踪观测进度
- 规划观测顺序
- 记录观测笔记

### 打开目标列表

- **快捷键**：Ctrl+L
- **工具栏**：点击列表图标
- **菜单**：视图 → 目标列表

## 添加目标

### 从星图添加

1. 在星图上右键点击天体
2. 选择"添加到目标列表"
3. 选择目标列表
4. 设置优先级和备注

### 从搜索添加

1. 搜索天体（Ctrl+F）
2. 在结果上右键
3. 选择"添加到目标列表"

### 批量添加

1. 点击"批量添加"
2. 输入天体列表（每行一个）
3. 点击"添加全部"

### 从文件导入

支持导入的格式：

- CSV 文件
- 文本文件
- N.I.N.A. 序列文件
- Stellarium 观测列表

## 目标信息

### 必填信息

- **天体标识**：名称或编号
- **坐标**：赤经和赤纬

### 可选信息

- **优先级**：高/中/低
- **状态**：待观测/进行中/已完成/已跳过
- **备注**：自定义备注
- **标签**：分类标签
- **计划时间**：预定观测时间

### 自动填充

系统自动填充的信息：

- 星等
- 类型
- 大小
- 今晚可见性
- 最佳观测时间

## 管理目标

### 编辑目标

1. 在列表中选择目标
2. 点击"编辑"或双击
3. 修改信息
4. 保存更改

### 删除目标

- 选中目标后按 Delete 键
- 右键选择"删除"
- 支持多选批量删除

### 更改状态

快速更改目标状态：

- **复选框**：点击标记完成
- **右键菜单**：选择状态
- **快捷键**：
  - C - 标记完成
  - S - 标记跳过
  - P - 标记进行中

### 设置优先级

- **拖拽排序**：拖动调整顺序
- **右键菜单**：设置优先级
- **快捷键**：
  - 1 - 高优先级
  - 2 - 中优先级
  - 3 - 低优先级

## 排序和筛选

### 排序选项

可按以下方式排序：

- **优先级**：按优先级排序
- **名称**：按字母顺序
- **赤经**：按赤经排序（观测顺序）
- **高度**：按当前高度角
- **可见时长**：按今晚可见时间
- **最佳时间**：按最佳观测时间
- **状态**：按完成状态

### 筛选选项

筛选显示的目标：

- **状态筛选**：只显示特定状态
- **类型筛选**：只显示特定类型
- **可见性筛选**：只显示当前可见
- **标签筛选**：按标签筛选

### 搜索目标

在列表中快速查找：

1. 按 Ctrl+F 打开搜索
2. 输入关键词
3. 匹配项高亮显示

## 多列表管理

### 创建列表

1. 点击"新建列表"
2. 输入列表名称
3. 设置列表属性

### 切换列表

- 使用下拉菜单切换
- 使用标签页切换
- 快捷键 Ctrl+1/2/3...

### 列表属性

每个列表可设置：

- **名称**：列表显示名称
- **描述**：列表说明
- **颜色**：标识颜色
- **默认排序**：默认排序方式

### 合并列表

合并多个列表：

1. 选择要合并的列表
2. 点击"合并列表"
3. 选择目标列表
4. 处理重复项

## 观测规划

### 生成观测计划

根据目标列表生成优化的观测顺序：

1. 选择要规划的目标
2. 点击"生成计划"
3. 设置规划参数
4. 查看优化结果

### 规划参数

- **开始时间**：观测开始时间
- **结束时间**：观测结束时间
- **最小高度**：最低观测高度
- **子午线规避**：避开子午线时段
- **优化目标**：效率/高度/顺序

### 查看计划

计划显示：

- 观测顺序
- 预计时间
- 高度变化
- 移动距离

## 可见性分析

### 单目标分析

选中目标查看：

- 今晚可见时段
- 最高高度和时间
- 月球距离
- 观测建议

### 批量分析

分析整个列表：

1. 点击"可见性分析"
2. 选择日期范围
3. 查看所有目标的可见性
4. 导出分析结果

### 可见性图表

图形化显示：

- 高度-时间曲线
- 多目标对比
- 最佳观测窗口

## 导入导出

### 导出列表

1. 选择要导出的列表
2. 点击"导出"
3. 选择格式和位置
4. 保存文件

### 支持的导出格式

- **CSV**：通用表格格式
- **JSON**：程序交换格式
- **PDF**：打印用报告
- **文本**：简单文本列表

### 导入列表

1. 点击"导入"
2. 选择文件
3. 映射字段（如需要）
4. 确认导入

### 兼容格式

可导入其他软件的列表：

- Stellarium 观测列表
- N.I.N.A. 高级序列器
- SGPro 目标列表
- APT 观测计划

## 与星图联动

### 定位目标

在星图上定位目标：

- 双击目标跳转到星图
- 右键选择"在星图显示"
- 显示视野框预览

### 批量标记

在星图上标记所有目标：

1. 启用"显示目标标记"
2. 目标位置显示标记
3. 点击标记查看详情

### 路径显示

显示观测路径：

- 连接各目标的路线
- 显示移动距离
- 优化建议

## 协作功能

### 分享列表

与他人分享目标列表：

1. 点击"分享"
2. 生成分享链接或文件
3. 发送给其他用户

### 导入分享列表

导入他人分享的列表：

1. 点击"导入"
2. 输入链接或选择文件
3. 预览并确认导入

## 常见问题

### 目标不显示

可能的原因：

1. 检查筛选条件
2. 检查是否在当前列表
3. 检查搜索框是否清空

### 无法添加目标

可能的原因：

1. 天体不在数据库中
2. 坐标格式错误
3. 列表已锁定

### 排序不正确

解决方法：

1. 检查排序设置
2. 刷新列表
3. 重新计算可见性
