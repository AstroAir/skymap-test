### 常见任务

#### 查找一个天体

1. 按 Ctrl+F 或点击搜索按钮
2. 输入天体名称（如"M31"）
3. 从结果列表选择
4. 双击定位到天体

#### 规划今晚的观测

1. 查看今晚推荐
2. 检查天气和月相
3. 创建观测列表
4. 计算最佳观测时间

#### 配置设备

1. 添加您的望远镜
2. 添加您的相机
3. 创建设备配置
4. 测试视野计算

## 键盘快捷键

### 常用快捷键

- **Ctrl+F** - 搜索天体
- **Ctrl+S** - 保存视图
- **T** - 跳转到当前时间
- **Space** - 暂停/继续时间
- **+/-** - 放大/缩小
- **方向键** - 旋转视角

更多快捷键请参考[附录 - 键盘快捷键](../appendix/keyboard-shortcuts.md)。
