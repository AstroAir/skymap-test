# 查看天体详情

本文档介绍如何查看和使用 SkyMap Test 中的天体详细信息。

## 打开详情面板

### 打开方式

- **双击天体**：在星图上双击天体
- **右键菜单**：右键选择"查看详情"
- **搜索结果**：双击搜索结果
- **快捷键**：选中天体后按 Enter

### 详情面板位置

详情面板可以：

- 右侧边栏显示（默认）
- 底部面板显示
- 独立弹窗显示
- 浮动窗口显示

## 基本信息

### 标识信息

- **主名称**：天体的主要名称
- **别名**：其他常用名称
- **目录编号**：各星表中的编号
- **中文名称**：中文译名或传统名称

### 位置信息

**赤道坐标**
- **赤经（RA）**：时分秒格式
- **赤纬（Dec）**：度分秒格式
- **历元**：J2000.0

**地平坐标**（实时计算）
- **方位角**：0°-360°
- **高度角**：-90°至+90°
- **时角**：距子午线的角度

**银河坐标**
- **银经**：银河坐标系经度
- **银纬**：银河坐标系纬度

### 物理信息

**恒星**
- **光谱类型**：如 G2V、M2III
- **温度**：表面有效温度
- **质量**：以太阳质量为单位
- **半径**：以太阳半径为单位
- **光度**：以太阳光度为单位

**深空天体**
- **类型**：星系、星云、星团等
- **大小**：角直径
- **距离**：到地球的距离
- **红移**：（星系）红移值

**太阳系天体**
- **距离**：当前到地球距离
- **相位**：照明百分比
- **角直径**：视角大小
- **轨道参数**：半长轴、偏心率等

## 观测信息

### 亮度信息

- **视星等**：从地球观测的亮度
- **绝对星等**：标准距离处的亮度
- **表面亮度**：每平方角秒亮度
- **色指数**：B-V 色指数

### 可见性信息

- **当前状态**：可见/不可见
- **升起时间**：今天升起的时间
- **落下时间**：今天落下的时间
- **中天时间**：过子午线的时间
- **最高高度**：今天的最高高度角

### 最佳观测

- **最佳时间**：建议的观测时间
- **最佳月份**：年度最佳观测期
- **观测难度**：入门/中级/高级
- **建议设备**：推荐的观测设备

## 图像信息

### 图像预览

详情面板显示天体图像：

- **DSS 图像**：数字化巡天图像
- **SDSS 图像**：斯隆数字巡天（如果有）
- **HST 图像**：哈勃望远镜图像（如果有）
- **用户图像**：用户上传的图像

### 图像操作

- **放大查看**：点击图像放大
- **切换来源**：切换不同图像源
- **下载图像**：保存到本地
- **在星图显示**：将图像叠加到星图

### 图像设置

- **自动加载**：打开详情时自动加载
- **图像质量**：选择分辨率
- **缓存图像**：离线使用

## 高度曲线

### 高度-时间图

显示天体高度随时间变化：

- **X轴**：时间（24小时）
- **Y轴**：高度角（-90°至+90°）
- **当前位置**：标记当前时刻
- **日出日落**：标记曙暮光时间

### 图表交互

- **悬停**：显示具体时间和高度
- **点击**：跳转到该时间
- **缩放**：放大查看细节
- **导出**：保存图表图像

### 多日期对比

比较不同日期的高度曲线：

1. 点击"添加日期"
2. 选择要比较的日期
3. 图表显示多条曲线
4. 不同颜色区分日期

## 拍摄信息

### 成像建议

针对天文摄影的建议：

- **建议曝光**：推荐的曝光时间
- **建议增益**：推荐的相机增益
- **滤镜选择**：推荐的滤镜
- **子帧数量**：建议的叠加帧数

### 视野匹配

检查设备视野匹配：

1. 选择当前设备组合
2. 显示天体在视野中的大小
3. 判断是否需要马赛克拼接
4. 生成拍摄计划

### 曝光计算器

计算最佳曝光参数：

- 输入天空背景亮度
- 输入设备参数
- 计算信噪比
- 优化曝光时间

## 相关天体

### 附近天体

显示周围的天体：

- 距离阈值可调
- 按距离排序
- 显示类型和亮度
- 点击可切换查看

### 同类天体

显示同类型的其他天体：

- 相似类型的深空天体
- 同星团的成员星
- 同星系群的星系

### 观测关联

与其他观测相关的天体：

- 附近的导星
- 可能的比较星
- 适合的校准源

## 外部资源

### 在线数据库

链接到外部资源：

- **SIMBAD**：天体数据库
- **NED**：河外星系数据库
- **VizieR**：天文目录服务
- **ADS**：天文文献数据库

### 图像资源

- **Sky-Map.org**：在线星图
- **WikiSky**：维基星空
- **Aladin**：交互式星图
- **ESASky**：欧空局星空

### 参考资料

- **维基百科**：百科条目
- **专业论文**：相关研究论文
- **观测报告**：业余观测报告

## 操作功能

### 添加到列表

将天体添加到观测列表：

1. 点击"添加到列表"
2. 选择目标列表
3. 设置优先级
4. 添加备注

### 创建标记

在星图上创建标记：

1. 点击"创建标记"
2. 选择标记样式
3. 添加标签文字
4. 保存标记

### 分享天体

分享天体信息：

- 复制坐标
- 生成链接
- 导出信息卡片
- 分享到社交媒体

### 报告问题

报告数据问题：

- 名称错误
- 坐标错误
- 缺失信息
- 图像问题

## 自定义显示

### 显示内容

选择显示哪些信息：

- 基本信息（始终显示）
- 物理参数
- 观测信息
- 图像
- 高度曲线
- 外部链接

### 面板布局

调整面板布局：

- 拖拽调整顺序
- 折叠不需要的区域
- 保存自定义布局

### 数据单位

选择数据显示单位：

- 距离：光年/秒差距/天文单位
- 角度：度分秒/十进制度
- 时间：UTC/本地时间
