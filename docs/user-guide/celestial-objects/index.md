# 天体对象

SkyMap Test 提供丰富的天体信息查询和显示功能。

## 支持的天体类型

### 恒星

- **亮星** - Hipparcos 星表中的恒星
- **双星** - 物理双星和光学双星
- **变星** - 各类变星（造父变星、食双星等）

### 深空天体 (DSO)

- **星系** - 椭圆星系、旋涡星系、不规则星系
- **星云** - 发射星云、反射星云、暗星云、行星状星云
- **星团** - 疏散星团、球状星团

### 太阳系天体

- **行星** - 八大行星
- **卫星** - 主要卫星
- **小行星** - 已编号小行星
- **彗星** - 周期彗星和新发现彗星

### 人造天体

- **卫星** - ISS、星链等人造卫星
- **空间站** - 国际空间站、天宫等

## 天体搜索

### 快速搜索

1. 点击搜索框或按 `Ctrl+F`
2. 输入天体名称或编号
3. 从结果列表中选择

### 支持的搜索格式

| 格式 | 示例 |
|------|------|
| 通用名称 | 猎户座大星云、仙女座星系 |
| Messier 编号 | M31、M42、M1 |
| NGC 编号 | NGC 7000、NGC 6992 |
| IC 编号 | IC 434、IC 1396 |
| HD 编号 | HD 48915（天狼星） |
| HIP 编号 | HIP 32349 |
| 拜耳命名 | α Ori、β And |

### 高级搜索

打开高级搜索对话框（`Ctrl+Shift+F`）：

- **类型筛选** - 按天体类型过滤
- **星等范围** - 设置亮度限制
- **大小范围** - 设置角直径限制
- **高度限制** - 仅显示当前可见目标

## 天体信息面板

### 打开方式

- 点击星图上的天体
- 在搜索结果中点击
- 从目标列表中选择

### 信息内容

#### 基本信息

- **名称** - 通用名称和目录编号
- **类型** - 天体分类
- **坐标** - 赤经、赤纬
- **星等** - 视星等、绝对星等

#### 物理参数

- **大小** - 角直径或实际尺寸
- **距离** - 到地球的距离
- **光谱类型** - 恒星光谱分类
- **红移** - 星系红移值

#### 观测信息

- **当前高度** - 实时高度角
- **方位角** - 当前方位
- **升落时间** - 今日升起和落下时间
- **中天时间** - 过中天时刻

### 扩展信息

点击 "更多信息" 获取：

- **SIMBAD 数据** - 专业天文数据库信息
- **维基百科** - 科普介绍
- **图片** - 专业拍摄图像
- **参考文献** - 相关研究论文

## 天体图库

### 查看图像

天体信息面板中点击图像标签：

- **DSS 图像** - 数字化巡天图像
- **专业照片** - 望远镜拍摄图像
- **用户照片** - 社区分享图像

### 图像操作

- **缩放** - 滚轮或双指缩放
- **平移** - 拖动移动
- **旋转** - 调整方向
- **反转** - 切换反色显示

## 天体名称翻译

### 多语言支持

天体名称支持多种语言显示：

- **中文** - 中国传统星名
- **英文** - 国际通用名称
- **拉丁文** - 科学命名
- **本地语言** - 当地传统名称

### 切换语言

1. 打开设置
2. 选择 "显示"
3. 设置 "天体名称语言"

## 收藏和标记

### 添加收藏

在天体信息面板中点击收藏图标：

- 收藏的天体会保存到本地
- 可以在收藏列表中快速访问
- 支持分组管理

### 添加到目标列表

点击 "添加到目标" 按钮：

- 快速添加到当前观测列表
- 设置优先级和备注
- 规划观测顺序
