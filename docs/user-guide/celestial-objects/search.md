# 搜索天体

本文档介绍如何使用 SkyMap Test 的天体搜索功能。

## 打开搜索

### 快捷方式

- **快捷键**：Ctrl+F 或 F3
- **工具栏**：点击搜索图标
- **菜单**：编辑 → 搜索天体

### 搜索界面

搜索界面包含：

- **搜索输入框**：输入搜索关键词
- **搜索类型选择**：选择搜索范围
- **高级筛选**：展开高级选项
- **结果列表**：显示匹配结果

## 基本搜索

### 按名称搜索

输入天体的名称进行搜索：

- **专有名称**：如"Betelgeuse"（参宿四）
- **中文名称**：如"北极星"、"仙女座星系"
- **拜耳命名**：如"α Ori"、"β Cyg"
- **弗兰斯蒂德编号**：如"61 Cyg"

### 按编号搜索

输入天体的目录编号：

- **梅西耶编号**：M1、M31、M42
- **NGC编号**：NGC 224、NGC 7000
- **IC编号**：IC 434、IC 1396
- **HD编号**：HD 48915
- **HIP编号**：HIP 32349

### 按坐标搜索

输入天体坐标定位：

- **赤道坐标**：`05h 34m 32s, +22° 00' 52"`
- **简化格式**：`05:34:32, +22:00:52`
- **度数格式**：`83.633, 22.014`

## 高级搜索

### 打开高级搜索

点击搜索框旁的"高级"按钮，或使用 Ctrl+Shift+F。

### 按类型筛选

选择要搜索的天体类型：

**恒星**
- 主序星
- 巨星
- 超巨星
- 白矮星
- 变星
- 双星

**深空天体**
- 星系
- 星云（发射、反射、行星状、暗星云）
- 星团（疏散、球状）
- 超新星遗迹

**太阳系天体**
- 行星
- 矮行星
- 小行星
- 彗星
- 卫星

### 按亮度筛选

设置星等范围：

- **最亮**：最小星等值（如 -1.0）
- **最暗**：最大星等值（如 6.0）
- **预设**：肉眼可见（< 6.0）、双筒可见（< 10.0）

### 按位置筛选

限制搜索的天区：

- **星座**：选择特定星座
- **赤经范围**：设置 RA 范围
- **赤纬范围**：设置 Dec 范围
- **当前视野**：只搜索当前可见区域

### 按可见性筛选

根据观测条件筛选：

- **今晚可见**：今晚能够观测到的天体
- **现在可见**：当前时刻在地平线以上
- **最佳观测**：处于最佳观测位置
- **拱极天体**：永不落下的天体

### 按距离筛选

限制天体距离：

- **太阳系内**：仅太阳系天体
- **近距离**：< 100 光年
- **银河系内**：< 100,000 光年
- **河外**：> 100,000 光年

## 搜索结果

### 结果列表

搜索结果显示：

- **天体名称**：主名称和别名
- **类型图标**：天体类型的图标
- **星等**：视星等
- **位置**：当前高度角
- **距离**：到地球的距离

### 排序方式

可按以下方式排序：

- **相关度**：按匹配程度排序（默认）
- **亮度**：按星等从亮到暗
- **高度**：按当前高度角
- **距离**：按到地球距离
- **名称**：按字母顺序

### 结果操作

对搜索结果的操作：

- **单击**：选中并显示简要信息
- **双击**：居中显示并打开详情
- **右键**：打开上下文菜单
- **添加到列表**：添加到观测目标列表

## 模糊搜索

### 智能匹配

搜索支持模糊匹配：

- **拼写纠错**：自动纠正常见拼写错误
- **部分匹配**：匹配名称的一部分
- **同义词**：匹配别名和译名
- **缩写展开**：识别常用缩写

### 搜索建议

输入时显示建议：

- 基于输入显示可能的匹配
- 显示最近搜索历史
- 显示热门搜索

## 快速搜索

### 命令模式

使用特殊前缀快速搜索：

| 前缀 | 功能 | 示例 |
|-----|------|-----|
| `m:` | 梅西耶天体 | `m:31` |
| `ngc:` | NGC天体 | `ngc:7000` |
| `ic:` | IC天体 | `ic:434` |
| `hd:` | HD星表 | `hd:48915` |
| `hip:` | 依巴谷星表 | `hip:32349` |
| `@` | 坐标 | `@05:34:32,+22:00:52` |

### 快速定位

搜索后快速定位：

- **Enter**：定位到第一个结果
- **Ctrl+Enter**：定位并打开详情
- **数字键**：定位到对应序号的结果

## 搜索历史

### 查看历史

- 点击搜索框时显示最近搜索
- 使用 ↑/↓ 键浏览历史
- 最多保存 100 条历史

### 管理历史

- **清除单条**：右键删除单条记录
- **清除全部**：在设置中清除所有历史
- **禁用历史**：关闭搜索历史记录

## 收藏夹

### 添加收藏

将常用搜索添加到收藏：

1. 执行搜索
2. 点击搜索框旁的星号图标
3. 输入收藏名称
4. 保存收藏

### 使用收藏

快速访问收藏的搜索：

- 点击收藏图标查看列表
- 点击收藏项执行搜索
- 支持导入/导出收藏

## 批量搜索

### 列表搜索

一次搜索多个天体：

1. 点击"批量搜索"
2. 输入天体列表（每行一个）
3. 点击搜索
4. 查看所有结果

### 导入列表

从文件导入搜索列表：

- 支持 CSV 格式
- 支持文本文件
- 自动识别天体名称

## 搜索设置

### 搜索选项

在设置中配置：

- **默认搜索类型**：设置默认范围
- **结果数量限制**：最大显示数量
- **自动完成**：启用/禁用建议
- **模糊匹配**：启用/禁用模糊搜索

### 性能设置

优化搜索性能：

- **索引预加载**：启动时加载索引
- **缓存结果**：缓存搜索结果
- **后台索引**：后台更新索引

## 常见问题

### 搜索没有结果

可能的原因：

1. 检查拼写是否正确
2. 尝试使用编号搜索
3. 扩大搜索范围
4. 检查筛选条件

### 搜索太慢

优化方法：

1. 缩小搜索范围
2. 使用更具体的关键词
3. 启用搜索缓存
4. 限制结果数量

### 找不到特定天体

解决方法：

1. 尝试不同的名称
2. 使用目录编号
3. 按坐标搜索
4. 检查是否在数据库中
