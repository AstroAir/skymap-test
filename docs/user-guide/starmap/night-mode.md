# 夜视模式

夜视模式（红光模式）帮助保护您在夜间观测时的暗适应能力。

## 什么是暗适应

### 暗适应原理

人眼在黑暗环境中需要约20-30分钟才能完全适应，这个过程称为暗适应：

- **视杆细胞**：负责暗视觉，对光线敏感
- **视紫红质**：暗适应的关键物质
- **恢复时间**：一旦暴露在亮光下，需要重新适应

### 红光的优势

红光对暗适应影响最小：

- 波长最长，能量最低
- 视杆细胞对红光不敏感
- 可以看清屏幕而不破坏暗适应

## 启用夜视模式

### 快捷方式

- **快捷键**：按 R 键切换
- **工具栏**：点击月亮图标
- **菜单**：视图 → 夜视模式

### 自动切换

可设置自动切换：

1. 打开设置 → 显示
2. 启用"自动夜视模式"
3. 设置触发条件：
   - 日落后自动启用
   - 根据系统时间
   - 根据环境光传感器

## 夜视模式设置

### 红光强度

调整红光的亮度：

| 级别 | 适用场景 |
|-----|---------|
| 最暗 | 极暗环境，最大保护 |
| 暗 | 正常野外观测 |
| 中等 | 有微弱环境光 |
| 亮 | 需要清晰阅读 |

### 颜色调整

自定义夜视颜色：

- **纯红**：#FF0000，最佳保护
- **深红**：#8B0000，柔和效果
- **琥珀**：#FF4500，略有橙色
- **自定义**：选择任意暗红色

### 对比度

调整夜视模式下的对比度：

- 提高对比度增强可读性
- 降低对比度减少眩光
- 推荐值：80-120%

## 界面元素

### 星图显示

夜视模式下的星图变化：

- 背景变为深红/黑色
- 恒星保持可见（红色或白色可选）
- 网格线变为暗红色
- 标签变为红色

### 控制面板

界面元素的调整：

- 按钮变为红色主题
- 文字变为红色
- 图标使用红色版本
- 背景变为深色

### 信息面板

天体信息显示：

- 文字颜色变红
- 图片可选灰度处理
- 图表使用红色配色

## 高级设置

### 亮度自动调节

根据环境自动调节：

1. 启用"自动亮度"
2. 设置最小/最大亮度
3. 设置调节速度

### 屏幕变暗

进一步降低屏幕亮度：

- **软件调光**：超越硬件最低亮度
- **滤镜叠加**：添加半透明黑色层
- **OLED 友好**：适合 OLED 屏幕

### 蓝光过滤

额外的蓝光过滤：

- 减少蓝光成分
- 进一步保护暗适应
- 可调节过滤强度

## 设备设置

### 硬件亮度

配合硬件设置：

1. 将屏幕亮度调到最低
2. 启用夜视模式
3. 使用软件调光进一步降低

### 外接显示器

外接显示器的注意事项：

- 确保显示器支持低亮度
- 使用遮光罩减少漏光
- 考虑使用红色滤光片

## 最佳实践

### 观测前准备

1. 观测前30分钟启用夜视模式
2. 调低屏幕亮度
3. 测试所有需要的功能
4. 确保软件配置完成

### 观测中使用

1. 尽量减少查看屏幕
2. 使用单眼查看（保护另一眼）
3. 快速查看后立即移开
4. 使用语音提示（如果支持）

### 其他注意事项

- 关闭不必要的通知
- 禁用弹窗提示
- 使用深色系统主题
- 关闭键盘背光

## 配合设备

### 红光手电

使用红光手电查看物品：

- 不要使用白光手电
- 红光手电不影响他人
- 建议使用可调光红光手电

### 设备屏幕

其他设备的建议：

- 手机启用红色滤镜
- 平板使用夜间模式
- 笔记本贴红色滤光膜

## 常见问题

### 夜视模式下看不清

解决方法：

1. 适当提高红光亮度
2. 增加对比度
3. 检查字体大小设置
4. 调整屏幕角度

### 红色太刺眼

解决方法：

1. 降低亮度
2. 选择更深的红色
3. 启用软件调光
4. 休息一下眼睛

### 无法完全变红

可能的原因：

- 某些元素不支持变色
- 第三方组件未适配
- 尝试刷新页面
