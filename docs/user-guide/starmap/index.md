# 星图概览

星图是 SkyMap Test 的核心功能，提供交互式的实时星空显示。

## 星图界面

### 主显示区

星图主显示区占据窗口的大部分区域，显示实时星空：

- **实时渲染**：根据当前时间实时更新星空
- **交互操作**：支持鼠标和键盘交互
- **平滑动画**：流畅的天体运动效果
- **多层显示**：同时显示多种类型的天体

### 状态栏

底部状态栏显示当前信息：

- **时间**：当前模拟时间
- **位置**：观测位置
- **坐标**：鼠标指向位置的坐标
- **FOV**：当前视场大小

## 星图图层

SkyMap Test 支持多种星图图层：

### 恒星层

显示恒星数据：

- **HIPPARCOS**：依巴谷星表，约11.8万颗恒星
- **Tycho-2**：第谷星表，约250万颗恒星
- **UCAC4**：美国海军天文台星表，超过1亿颗恒星

### 深空天体层

显示深空天体：

- **梅西耶天体**：M1-M110
- **NGC天体**：NGC星云星团表
- **IC天体**：IC索引星表
- **其他深空天体**：Sh2、vdB等

### 太阳系天体

显示太阳系天体：

- **太阳**：位置和相位
- **月亮**：位置、相位、亮度
- **行星**：八大行星的位置和亮度
- **矮行星**：冥王星、谷神星等

### 人造卫星

显示人造卫星（需要网络连接）：

- **位置**：实时位置
- **轨迹**：运行轨迹
- **可见性**：可见时段

## 星图数据来源

### 在线数据

联网时可获取在线数据：

- **HiPS服务**：高清天文图像
- **DSS**：数字化巡天数据
- **SDSS**：斯隆数字巡天
- **其他服务**：各种专业天文数据

### 离线数据

离线时可使用缓存数据：

- **自动缓存**：浏览过的区域自动缓存
- **手动下载**：下载特定区域数据
- **缓存管理**：管理和清理缓存

## 星图渲染质量

### 渲染模式

多种渲染模式可选：

- **性能模式**：流畅度优先，适合旧设备
- **平衡模式**：性能和质量平衡
- **质量模式**：画质优先，需要较好硬件
- **自定义模式**：手动调整各项参数

### 星等限制

设置显示的最暗星等：

- **推荐设置**：
  - 城市观测：4.0-5.0等
  - 郊区观测：5.5-6.5等
  - 山区观测：7.0等以上
- **自动调整**：根据缩放级别自动调整

### 天体大小

调整天体显示大小：

- **真实大小**：按实际大小比例显示
- **对数缩放**：使用对数缩放显示
- **固定大小**：所有天体使用固定大小
- **自定义**：手动调整大小比例

## 辅助显示

### 坐标网格

显示坐标网格帮助定位：

- **地平网格**：显示方位角和高度角
- **赤道网格**：显示赤经和赤纬
- **银河网格**：显示银河坐标系
- **黄道网格**：显示黄道坐标系

### 星座元素

显示星座相关元素：

- **星座连线**：连接主要恒星形成图案
- **星座边界**：显示星座的官方边界
- **星座名称**：显示星座的拉丁名和中文译名
- **星座图像**：显示星座的艺术图像

### 参考线和点

重要的参考线和点：

- **天顶**：观测者正上方的点
- **天极**：北天极和南天极
- **黄道**：太阳在天球上的路径
- **银河**：银河系平面
- **赤道**：天赤道
- **本初子午线**：赤经零点

## 天体标签

### 标签内容

可显示的天体标签：

- **恒星名称**：专有名称和拜耳 designation
- **深空天体**：梅西耶、NGC编号
- **行星名称**：行星名称
- **天体坐标**：赤经赤纬坐标
- **星等**：天体亮度
- **距离**：天体距离

### 标签样式

自定义标签样式：

- **字体大小**：调整标签字体大小
- **标签颜色**：设置标签颜色
- **标签背景**：添加半透明背景
- **标签密度**：控制标签数量

## 时间控制

### 时间模式

多种时间模式：

- **实时模式**：显示当前时刻的天空
- **手动模式**：手动设置日期和时间
- **动画模式**：时间自动流逝

### 时间流速

控制时间流逝速度：

- **暂停**：时间停止
- **慢速**：1秒=1分钟
- **中速**：1秒=1小时
- **快速**：1秒=1天
- **超快**：1秒=1年

### 时间跳转

快速跳转到特定时间：

- **现在**：跳转到当前时间
- **日落**：跳转到日落时间
- **日出**：跳转到日出时间
- **天文昏影终**：跳转到天文昏影结束
- **自定义**：输入特定时间
