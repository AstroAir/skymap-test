# 显示设置

本文档介绍如何自定义 SkyMap Test 的星图显示效果。

## 访问设置

打开显示设置的方式：

- 点击右上角齿轮图标
- 使用快捷键 Ctrl+,
- 菜单：视图 → 显示设置

## 天体显示

### 恒星设置

#### 星等限制

控制显示的最暗恒星：

| 星等 | 可见恒星数 | 适用场景 |
|-----|----------|---------|
| 4.0 | ~500 | 城市观测 |
| 5.0 | ~1,600 | 郊区观测 |
| 6.0 | ~5,000 | 农村地区 |
| 7.0 | ~15,000 | 山区暗天 |
| 8.0+ | ~40,000+ | 专业观测 |

#### 恒星大小

- **真实比例**：按亮度比例显示大小
- **对数缩放**：使用对数比例，突出暗星
- **固定大小**：所有恒星相同大小
- **自定义**：手动设置大小系数

#### 恒星颜色

- **真实颜色**：根据光谱类型显示颜色
- **单色**：所有恒星使用相同颜色
- **增强对比**：增强颜色差异
- **色温映射**：按温度映射颜色

### 深空天体

#### DSO 显示选项

- **显示 DSO**：启用/禁用深空天体
- **星等限制**：设置最暗 DSO 星等
- **显示轮廓**：显示天体的实际大小轮廓
- **显示图像**：显示天体的实际图像

#### DSO 类型过滤

选择显示的天体类型：

- 星系
- 星云（发射、反射、行星状）
- 星团（疏散、球状）
- 其他（暗星云、超新星遗迹等）

### 太阳系天体

#### 行星显示

- **显示行星**：启用/禁用行星
- **显示轨道**：显示行星轨道
- **显示标签**：显示行星名称
- **相位显示**：显示行星相位（金星、火星等）

#### 月球显示

- **显示月球**：启用/禁用月球
- **显示相位**：显示当前月相
- **显示细节**：显示月球表面细节
- **亮度模拟**：模拟月光对天空的影响

#### 太阳显示

- **显示太阳**：启用/禁用太阳
- **曙暮光区域**：显示不同的曙暮光带

### 人造卫星

- **显示卫星**：启用/禁用卫星
- **显示轨迹**：显示卫星轨道
- **闪烁效果**：模拟铱星闪烁
- **分类过滤**：按卫星类型过滤

## 坐标网格

### 地平坐标系

- **显示网格**：启用地平坐标网格
- **网格密度**：调整网格线密度
- **颜色设置**：自定义网格颜色
- **显示标签**：显示坐标值

### 赤道坐标系

- **显示网格**：启用赤道坐标网格
- **显示赤道**：突出显示天赤道
- **显示极点**：标记天极位置

### 银河坐标系

- **显示网格**：启用银河坐标网格
- **显示银道面**：突出显示银河平面

### 黄道坐标系

- **显示网格**：启用黄道坐标网格
- **显示黄道**：突出显示黄道
- **显示黄道十二宫**：显示黄道星座区域

## 星座元素

### 星座连线

- **显示连线**：启用星座连线
- **连线样式**：实线、虚线、点线
- **连线颜色**：自定义颜色
- **连线粗细**：调整线宽

### 星座边界

- **显示边界**：显示 IAU 官方边界
- **边界样式**：虚线显示
- **边界颜色**：淡色避免干扰

### 星座名称

- **显示名称**：启用星座名称
- **语言选择**：
  - 拉丁名（Orion）
  - 中文名（猎户座）
  - 缩写（Ori）
- **字体大小**：调整名称字体

### 星座艺术

- **显示艺术图**：显示星座神话图像
- **透明度**：调整图像透明度
- **艺术风格**：选择不同风格

## 参考线

### 地平线

- **显示地平线**：启用地平线
- **地平线样式**：直线或地形轮廓
- **地平线颜色**：自定义颜色

### 子午线

- **显示子午线**：显示本地子午线
- **显示时角线**：显示特定时角

### 黄道

- **显示黄道**：显示太阳路径
- **显示黄极**：标记黄道极点

### 银河

- **显示银河**：显示银河带
- **银河亮度**：调整显示强度
- **银河细节**：显示暗带和结构

## 天空背景

### 背景颜色

- **日间天空**：模拟白天天空
- **夜间天空**：深蓝色或黑色
- **曙暮光渐变**：自动过渡颜色

### HiPS Survey

选择天空背景图像：

| Survey | 描述 | 分辨率 |
|--------|-----|-------|
| DSS | 数字化巡天 | 中等 |
| DSS2 Color | 彩色版本 | 高 |
| SDSS | 斯隆数字巡天 | 很高 |
| 2MASS | 红外巡天 | 高 |
| WISE | 宽视场红外 | 中等 |

### 大气效果

- **大气散射**：模拟大气散射效果
- **消光效应**：模拟地平线附近的消光
- **折射校正**：补偿大气折射

## 标签显示

### 标签内容

选择显示的信息：

- 天体名称
- 星等
- 坐标
- 距离
- 光谱类型

### 标签样式

- **字体大小**：小/中/大
- **字体颜色**：白色/自动/自定义
- **背景**：无/半透明/不透明
- **位置**：上/下/左/右

### 标签密度

- **稀疏**：只显示亮星标签
- **中等**：显示主要天体
- **密集**：显示所有可见天体

## 预设配置

### 内置预设

- **默认**：平衡的默认设置
- **观测模式**：优化实际观测
- **摄影模式**：优化天文摄影
- **演示模式**：适合展示和教学
- **简洁模式**：最小化干扰

### 保存自定义预设

1. 调整好所有设置
2. 点击"保存为预设"
3. 输入预设名称
4. 从预设列表快速切换

## 性能设置

### 渲染质量

- **低**：适合旧设备
- **中**：推荐设置
- **高**：高端设备
- **最高**：专业显卡

### 帧率限制

- **30 FPS**：节能模式
- **60 FPS**：标准模式
- **无限制**：最流畅
