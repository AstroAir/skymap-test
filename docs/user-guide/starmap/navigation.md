# 导航控制

本文档介绍如何使用 SkyMap Test 的星图导航功能。

## 视角控制

### 鼠标控制

#### 旋转视角

- **左键拖动**：旋转星图视角
- **按住 Shift + 左键拖动**：慢速精确旋转

#### 缩放

- **滚轮滚动**：放大/缩小视图
- **按住 Ctrl + 滚轮**：精细缩放

#### 点击操作

- **单击天体**：选中天体，显示简要信息
- **双击天体**：居中显示并打开详情面板
- **右键点击**：打开上下文菜单

### 键盘控制

#### 方向键

- **↑ / ↓**：上下旋转视角
- **← / →**：左右旋转视角
- **Page Up / Page Down**：快速上下旋转

#### 缩放快捷键

- **+ / =**：放大
- **- / _**：缩小
- **Home**：重置为默认视角
- **End**：最大广角视图

#### 功能快捷键

- **N**：朝向北方
- **S**：朝向南方
- **E**：朝向东方
- **W**：朝向西方
- **Z**：朝向天顶

### 触摸控制

在支持触摸的设备上：

- **单指拖动**：旋转视角
- **双指缩放**：放大/缩小
- **双指旋转**：旋转视图方向
- **双击**：居中选中的天体

## 视角模式

### 地平视角

默认视角模式，模拟站在地面观测：

- 地平线始终水平
- 方位角和高度角直观显示
- 适合实际观测参考

### 赤道视角

以天赤道为参考的视角：

- 天赤道水平显示
- 便于查找赤道坐标
- 适合天文摄影构图

### 自由视角

不受任何约束的自由视角：

- 可任意旋转
- 适合探索全天球
- 可能导致方向迷失

## 定位功能

### 搜索定位

1. 按 **Ctrl+F** 打开搜索框
2. 输入天体名称或编号
3. 选择搜索结果
4. 视图自动居中到目标

### 坐标定位

1. 按 **Ctrl+G** 打开坐标输入
2. 输入赤经/赤纬或方位角/高度角
3. 点击"定位"按钮
4. 视图跳转到指定坐标

### 快速定位

- **天顶**：按 Z 键
- **北天极**：按 Ctrl+N
- **南天极**：按 Ctrl+S
- **银河中心**：按 Ctrl+G, C

## 视野控制

### 视野大小

视野（FOV）控制可见范围：

| 视野角度 | 适用场景 |
|---------|---------|
| 120° | 全景浏览 |
| 60° | 星座观察 |
| 10° | 天体群观察 |
| 1° | 单个天体 |
| 0.1° | 深空细节 |

### 视野预设

快速切换到预设视野：

- **F1**：广角视野（90°）
- **F2**：标准视野（60°）
- **F3**：中等视野（30°）
- **F4**：窄视野（10°）
- **F5**：望远镜视野（1°）

### 设备视野

根据设备计算真实视野：

1. 打开设备管理器
2. 选择望远镜和相机
3. 点击"显示视野框"
4. 视野框显示在星图上

## 跟踪模式

### 恒星时跟踪

自动补偿地球自转：

- 保持天体在视野中心
- 模拟赤道仪跟踪效果
- 适合长时间观测规划

### 天体跟踪

锁定跟踪特定天体：

1. 选中目标天体
2. 右键选择"跟踪此天体"
3. 视图随天体移动
4. 按 Esc 解除跟踪

### 太阳系天体跟踪

跟踪太阳系天体的视运动：

- 自动补偿天体的本身运动
- 适合行星、月球等观测
- 显示轨迹预测

## 视图历史

### 历史导航

- **Alt + ←**：返回上一视图
- **Alt + →**：前进到下一视图
- 最多保存50个历史记录

### 视图书签

保存常用视图：

1. 调整到想要的视角
2. 按 Ctrl+B 添加书签
3. 输入书签名称
4. 从书签列表快速恢复

## 多显示器支持

### 扩展显示

- 将星图窗口拖到第二显示器
- 支持不同分辨率
- 独立的视角控制

### 全屏模式

- 按 F11 切换全屏
- 隐藏所有界面元素
- 按 Esc 退出全屏

## 性能设置

### 平滑滚动

调整视角变化的平滑度：

- **关闭**：即时响应，可能卡顿
- **低**：轻微平滑
- **中**：适中平滑（推荐）
- **高**：非常平滑，略有延迟

### 惯性滚动

拖动释放后的惯性效果：

- 启用后拖动会有惯性
- 可调整惯性强度
- 单击可立即停止

## 常见问题

### 视角卡住不动

1. 检查是否启用了跟踪模式
2. 按 Esc 解除跟踪
3. 按 Home 重置视角

### 缩放不工作

1. 检查鼠标滚轮设置
2. 尝试使用 +/- 键
3. 检查视野是否已到极限

### 方向迷失

1. 按 N 朝向北方
2. 按 Home 重置视角
3. 启用坐标网格辅助定位
