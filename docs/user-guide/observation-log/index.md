# 观测日志

SkyMap Test 提供观测日志功能，帮助您记录和回顾观测活动。

## 功能概览

### 核心功能

- **会话管理** - 创建和管理观测会话
- **观测记录** - 记录每个目标的观测详情
- **条件记录** - 记录天气和观测条件
- **统计分析** - 查看观测历史统计

### 记录内容

- 观测时间和位置
- 观测目标信息
- 使用的设备
- 天气和视宁度
- 个人笔记和评分
- 附加图像

## 观测会话

### 创建会话

1. 点击 "开始观测会话"
2. 选择观测位置
3. 记录初始条件（可选）
4. 开始添加观测记录

### 会话信息

每个会话包含：

- **开始时间** - 会话开始时刻
- **结束时间** - 会话结束时刻
- **位置** - 观测地点
- **条件** - 天气、视宁度等
- **观测列表** - 本次观测的目标

### 结束会话

1. 点击 "结束会话"
2. 添加会话总结（可选）
3. 保存并关闭

## 添加观测记录

### 快速添加

在观测天体时：

1. 点击天体信息面板的 "记录观测"
2. 自动填充天体信息
3. 添加笔记和评分
4. 保存记录

### 详细记录

打开观测日志面板：

1. 点击 "添加记录"
2. 选择或搜索天体
3. 填写详细信息：
   - 观测时间
   - 使用设备
   - 放大倍率
   - 观测笔记
   - 评分（1-5星）
4. 添加图像（可选）
5. 保存

### 记录字段

| 字段 | 说明 |
|------|------|
| 目标 | 观测的天体 |
| 时间 | 观测时刻 |
| 设备 | 使用的望远镜/目镜 |
| 评分 | 1-5 星评级 |
| 笔记 | 观测体验描述 |
| 图像 | 附加照片 |

## 观测条件

### 记录条件

在会话中记录观测条件：

- **视宁度** - 1-5 级（差到优秀）
- **透明度** - 1-5 级
- **温度** - 当前气温
- **湿度** - 空气湿度
- **云量** - 云层覆盖程度
- **光污染** - Bortle 等级

### 条件变化

可以在会话中多次记录条件变化：

1. 点击 "更新条件"
2. 记录新的条件值
3. 系统自动记录时间戳

## 查看历史

### 会话列表

打开观测日志查看历史会话：

- 按日期排序
- 显示会话摘要
- 快速预览观测目标

### 筛选和搜索

- **按日期** - 选择日期范围
- **按位置** - 筛选特定观测点
- **按目标** - 搜索特定天体
- **按评分** - 筛选高评分观测

### 详情查看

点击会话查看详情：

- 完整的观测记录列表
- 条件变化时间线
- 统计信息

## 统计分析

### 观测统计

查看您的观测数据统计：

- **总观测次数** - 累计观测记录数
- **观测天数** - 有观测的日期数
- **目标数量** - 观测过的不同目标
- **总时长** - 累计观测时间

### 目标分布

图表显示：

- 按类型分布（星系、星云、星团等）
- 按星座分布
- 按月份分布

### 热门目标

显示观测次数最多的目标。

## 导入导出

### 导出日志

1. 打开观测日志
2. 选择要导出的会话
3. 点击 "导出"
4. 选择格式（JSON/CSV/PDF）
5. 保存文件

### 导入日志

1. 点击 "导入"
2. 选择日志文件
3. 预览数据
4. 确认导入

## 快捷操作

| 操作 | 快捷键 |
|------|--------|
| 打开观测日志 | `Ctrl+J` |
| 开始新会话 | `Ctrl+Shift+N` |
| 快速记录当前目标 | `Ctrl+Shift+R` |
| 结束当前会话 | `Ctrl+Shift+E` |
