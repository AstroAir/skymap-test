# 天空标记

SkyMap Test 支持在星图上添加自定义标记，帮助您记录和分享观测位置。

## 功能概览

### 标记类型

- **位置标记** - 标记天空中的特定位置
- **区域标记** - 标记一个天区范围
- **路径标记** - 标记移动轨迹

### 标记属性

每个标记包含：

- **名称** - 标记的描述性名称
- **坐标** - 赤经/赤纬位置
- **颜色** - 标记显示颜色
- **图标** - 标记图标样式
- **备注** - 附加说明文字
- **分组** - 所属标记组

## 使用指南

### 添加标记

#### 方式一：右键菜单

1. 在星图上右键点击目标位置
2. 选择 "添加标记"
3. 输入标记名称
4. 选择颜色和图标
5. 点击保存

#### 方式二：精确坐标

1. 打开标记管理器（`Ctrl+M`）
2. 点击 "新建标记"
3. 输入精确的赤经/赤纬坐标
4. 填写其他属性
5. 保存标记

### 编辑标记

1. 点击星图上的标记
2. 在弹出面板中点击编辑
3. 修改标记属性
4. 保存更改

或者：

1. 打开标记管理器
2. 找到要编辑的标记
3. 点击编辑图标
4. 修改并保存

### 删除标记

- 在标记信息面板中点击删除
- 在标记管理器中批量删除
- 右键标记选择删除

### 移动标记

1. 点击选中标记
2. 按住 `Shift` 拖动到新位置
3. 释放鼠标确认

## 标记分组

### 创建分组

1. 打开标记管理器
2. 点击 "新建分组"
3. 输入分组名称
4. 选择默认颜色

### 管理分组

- **重命名** - 双击分组名称
- **更改颜色** - 右键选择颜色
- **删除分组** - 右键删除（标记移至默认组）

### 批量操作

- 选中多个标记后可批量：
  - 移动到分组
  - 更改颜色
  - 显示/隐藏
  - 删除

## 显示控制

### 显示/隐藏标记

- **全部标记** - 工具栏标记按钮
- **按分组** - 在标记管理器中切换
- **单个标记** - 点击标记的可见性图标

### 标记样式

可自定义标记的显示样式：

- **大小** - 标记图标大小
- **透明度** - 标记透明度
- **标签** - 是否显示名称标签
- **连接线** - 是否显示指向线

## 导入导出

### 导出标记

1. 打开标记管理器
2. 选择要导出的标记或分组
3. 点击 "导出"
4. 选择格式（JSON/CSV）
5. 保存文件

### 导入标记

1. 打开标记管理器
2. 点击 "导入"
3. 选择标记文件
4. 预览并确认导入

### 支持格式

- **JSON** - 完整标记数据
- **CSV** - 简单坐标列表
- **Stellarium** - Stellarium 书签格式

## 应用场景

### 观测记录

- 标记已观测过的目标
- 记录有趣的天区
- 标注观测计划位置

### 摄影规划

- 标记拍摄中心点
- 规划马赛克覆盖范围
- 记录最佳取景位置

### 学习研究

- 标记正在学习的星座
- 记录特殊天象位置
- 标注研究目标

## 快捷操作

| 操作 | 快捷键 |
|------|--------|
| 打开标记管理器 | `Ctrl+M` |
| 在当前位置添加标记 | `M` |
| 显示/隐藏所有标记 | `Shift+M` |
| 删除选中标记 | `Delete` |
