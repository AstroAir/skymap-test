# 设备管理

SkyMap Test 提供完善的设备管理功能，帮助您配置和管理天文观测设备。

## 功能概览

### 支持的设备类型

- **望远镜** - 折射、反射、折反射等各类望远镜
- **相机** - CCD、CMOS 天文相机和 DSLR
- **目镜** - 各种焦距的目镜
- **增倍镜/减焦镜** - Barlow 镜和减焦镜
- **滤镜** - 光害滤镜、窄带滤镜等

### 核心功能

- **设备配置** - 添加和配置您的设备参数
- **视野计算** - 自动计算望远镜+相机组合的视野
- **预设管理** - 内置常见设备预设
- **设备组合** - 快速切换不同设备配置

## 设备管理器

### 打开方式

点击工具栏的设备图标或使用快捷键 `Ctrl+E` 打开设备管理器。

### 功能面板

设备管理器包含以下标签页：

1. **望远镜** - 管理望远镜配置
2. **相机** - 管理相机配置
3. **目镜** - 管理目镜配置
4. **附件** - 管理增倍镜、减焦镜、滤镜

### 添加设备

1. 点击 "+" 按钮
2. 选择从预设添加或手动输入
3. 填写设备参数
4. 保存配置

## 视野模拟

### FOV 模拟器

根据设备配置在星图上显示实际视野：

- **单帧视野** - 显示相机成像范围
- **马赛克模式** - 规划多帧拼接
- **旋转调整** - 调整视野旋转角度
- **目镜视野** - 模拟目视观测视野

### 视野计算

系统自动计算：

- 视野角度（度/分）
- 像素比例（角秒/像素）
- 放大倍数
- 分辨率限制

## 内置预设

### 望远镜预设

系统内置常见望远镜品牌和型号：

- Celestron（星特朗）
- Sky-Watcher（信达）
- Meade
- Takahashi（高桥）
- William Optics
- 更多...

### 相机预设

系统内置常见天文相机：

- ZWO ASI 系列
- QHY 系列
- Atik 系列
- 常见 DSLR 型号
