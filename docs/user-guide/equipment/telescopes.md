# 望远镜配置

本文档介绍如何在 SkyMap Test 中配置和管理望远镜。

## 望远镜管理

### 打开设备管理器

- **工具栏**：点击设备图标
- **菜单**：工具 → 设备管理器
- **快捷键**：Ctrl+E

### 望远镜列表

设备管理器显示所有已添加的望远镜：

- 望远镜名称
- 类型和规格
- 默认标记
- 操作按钮

## 添加望远镜

### 基本信息

添加新望远镜时需要输入：

**必填信息**
- **名称**：望远镜的识别名称
- **口径**：主镜口径（毫米）
- **焦距**：有效焦距（毫米）

**可选信息**
- **类型**：光学系统类型
- **品牌**：制造商
- **型号**：产品型号
- **序列号**：设备序列号

### 望远镜类型

选择望远镜的光学类型：

| 类型 | 描述 | 特点 |
|-----|------|------|
| 折射式 | 透镜系统 | 锐利、免维护 |
| 反射式 | 反射镜系统 | 大口径、经济 |
| 折反式 | 混合系统 | 紧凑、便携 |

#### 折射式望远镜

- 消色差折射镜（Achromatic）
- 复消色差折射镜（Apochromatic/APO）
- ED折射镜

#### 反射式望远镜

- 牛顿式（Newtonian）
- 道布森式（Dobsonian）
- 里奇-克莱琴式（Ritchey-Chrétien）

#### 折反式望远镜

- 施密特-卡塞格林（Schmidt-Cassegrain）
- 马克苏托夫-卡塞格林（Maksutov-Cassegrain）
- 马克苏托夫-牛顿（Maksutov-Newtonian）

### 高级参数

**光学参数**
- **焦比（F/ratio）**：自动计算或手动输入
- **中央遮挡**：副镜遮挡比例
- **集光力**：相对于肉眼的倍数

**机械参数**
- **镜筒长度**：物理长度
- **重量**：设备重量
- **后截距**：焦点到接口的距离

## 编辑望远镜

### 修改参数

1. 在列表中选择望远镜
2. 点击"编辑"按钮
3. 修改需要更改的参数
4. 点击"保存"

### 复制望远镜

基于现有望远镜创建新配置：

1. 选择要复制的望远镜
2. 点击"复制"
3. 修改名称和参数
4. 保存为新望远镜

### 删除望远镜

1. 选择要删除的望远镜
2. 点击"删除"
3. 确认删除操作

## 默认望远镜

### 设置默认

将常用望远镜设为默认：

1. 选择望远镜
2. 点击"设为默认"
3. 默认望远镜显示星号标记

### 默认望远镜用途

默认望远镜自动用于：

- 视野计算
- 放大倍数计算
- 极限星等估算
- 观测规划建议

## 视野计算

### 与相机配合

选择望远镜和相机后自动计算：

- **视野大小**：角度范围
- **像素比例**：角秒/像素
- **采样率**：与大气视宁度对比

### 与目镜配合

计算目视观测参数：

```
放大倍数 = 望远镜焦距 / 目镜焦距
出瞳直径 = 望远镜口径 / 放大倍数
真实视野 = 目镜表观视野 / 放大倍数
```

### 视野框显示

在星图上显示视野框：

1. 选择设备组合
2. 启用"显示视野框"
3. 视野框显示在星图中心
4. 可拖动视野框到目标位置

## 性能计算

### 分辨率

理论分辨率（道威斯极限）：

```
分辨率（角秒）= 116 / 口径（毫米）
```

### 极限星等

理论极限星等：

```
极限星等 ≈ 2.7 + 5 × log10(口径mm)
```

| 口径 | 极限星等 |
|-----|---------|
| 70mm | 11.9 |
| 100mm | 12.7 |
| 150mm | 13.6 |
| 200mm | 14.2 |
| 300mm | 15.1 |

### 最佳放大倍数

- **最低有效**：口径(mm) / 6 至 7
- **最佳观测**：口径(mm) × 1 至 1.5
- **理论最高**：口径(mm) × 2 至 2.5

## 减焦镜和增焦镜

### 添加减焦镜

减焦镜可缩短有效焦距：

1. 在望远镜详情中点击"添加附件"
2. 选择"减焦镜"
3. 输入减焦比（如 0.63x、0.8x）
4. 保存配置

### 添加增焦镜

巴洛镜可延长有效焦距：

1. 添加"增焦镜/巴洛镜"
2. 输入增焦比（如 2x、3x）
3. 保存配置

### 配置切换

快速切换不同配置：

- 创建多个预设
- 一键切换配置
- 自动更新视野计算

## 望远镜预设

### 常用望远镜数据库

1. 点击"从预设添加"
2. 搜索品牌或型号
3. 选择匹配的型号
4. 参数自动填充

## 导入导出

### 导出配置

备份望远镜配置：

1. 选择要导出的望远镜
2. 点击"导出"
3. 选择保存位置
4. 保存为 JSON 文件

### 导入配置

从文件导入望远镜：

1. 点击"导入"
2. 选择配置文件
3. 确认导入
4. 望远镜添加到列表

### 批量操作

- 全选导出所有设备
- 批量删除多个设备
- 从其他软件导入

## 与赤道仪配合

### 赤道仪配置

配置望远镜使用的赤道仪：

- 赤道仪型号

- 跟踪精度

### 平衡计算

计算设备平衡：

- 配重需求

## 常见问题

### 焦距不确定

如何确定焦距：

1. 查看望远镜铭牌
2. 查阅产品说明书
3. 使用星点法测量
4. 联系制造商确认

### 口径测量

测量实际口径：

- 测量主镜直径（反射式）
- 测量物镜直径（折射式）
- 注意排除镜筒内径

### 视野计算不准

可能的原因：

1. 焦距参数不准确
2. 使用了减焦/增焦镜
3. 相机参数错误
4. 需要校准
