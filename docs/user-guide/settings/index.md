# 应用设置

SkyMap Test 提供丰富的自定义设置选项，让您根据个人偏好调整应用。

## 设置面板

### 打开方式

- 点击工具栏的设置图标
- 使用快捷键 `Ctrl+,`
- 菜单栏 → 设置

### 设置分类

- **通用** - 基本应用设置
- **显示** - 星图显示选项
- **位置** - 观测位置设置
- **设备** - 设备配置
- **网络** - 网络和缓存
- **快捷键** - 键盘快捷键
- **高级** - 高级选项

## 通用设置

### 语言

- **界面语言** - 中文 / English
- **天体名称** - 选择天体名称显示语言

### 启动行为

- **启动时打开** - 选择默认打开的页面
- **恢复上次状态** - 记住上次的视图设置
- **自动检查更新** - 启动时检查新版本

### 数据目录

- **数据存储位置** - 自定义数据存储路径
- **打开数据目录** - 在文件管理器中打开

## 显示设置

### 星图显示

- **星等限制** - 最暗显示的恒星星等
- **恒星大小** - 恒星显示大小比例
- **DSO 显示** - 显示深空天体
- **行星显示** - 显示太阳系行星

### 坐标网格

- **赤道坐标网格** - 显示赤经/赤纬网格
- **地平坐标网格** - 显示高度/方位网格
- **网格间距** - 网格线间距
- **网格颜色** - 自定义网格颜色

### 星座

- **星座连线** - 显示星座连线
- **星座边界** - 显示星座边界
- **星座名称** - 显示星座名称
- **星座艺术图** - 显示星座神话图案

### 天体标签

- **恒星名称** - 显示亮星名称
- **DSO 名称** - 显示深空天体名称
- **行星名称** - 显示行星名称
- **标签大小** - 调整标签字体大小

### 其他显示

- **银河** - 显示银河带
- **黄道** - 显示黄道
- **子午线** - 显示子午线
- **地平线** - 显示地平线

## 位置设置

### 默认位置

- **当前位置** - 显示和切换当前位置
- **自动定位** - 启动时自动获取位置
- **位置管理** - 打开位置管理器

### 时间设置

- **时区** - 设置时区（自动/手动）
- **时间格式** - 12 小时制 / 24 小时制
- **日期格式** - 选择日期显示格式

## 网络设置

### 在线服务

- **启用在线查询** - 允许网络数据请求
- **图像服务** - 启用天体图像加载
- **信息服务** - 启用 SIMBAD/维基百科查询

### 代理设置

- **使用代理** - 启用代理服务器
- **代理地址** - 代理服务器 URL
- **代理认证** - 用户名和密码

### 缓存设置

- **缓存大小** - 最大缓存空间
- **缓存位置** - 缓存存储路径
- **清理缓存** - 手动清理缓存

## 快捷键设置

### 自定义快捷键

可以自定义以下操作的快捷键：

- 导航快捷键
- 搜索快捷键
- 显示切换快捷键
- 工具快捷键

### 恢复默认

点击 "恢复默认" 将所有快捷键重置为默认值。

## 高级设置

### 性能

- **渲染质量** - 低/中/高
- **动画效果** - 启用/禁用动画
- **硬件加速** - 使用 GPU 加速

### 开发者选项

- **显示 FPS** - 显示帧率
- **调试模式** - 启用调试日志
- **控制台** - 打开开发者控制台

### 数据管理

- **导出设置** - 导出所有设置
- **导入设置** - 从文件导入设置
- **重置设置** - 恢复所有默认设置

## 主题设置

### 主题模式

- **浅色模式** - 白天使用
- **深色模式** - 夜间使用
- **跟随系统** - 自动跟随系统设置

### 夜间模式

专为天文观测设计的红色界面：

- **启用夜间模式** - 切换红色界面
- **亮度** - 调整屏幕亮度
- **红色强度** - 调整红色滤镜强度

### 自定义主题

- **主题颜色** - 自定义主题色
- **背景颜色** - 自定义背景色
- **预设主题** - 选择预设主题方案

## 设置同步

### 云同步

- **启用同步** - 开启设置云同步
- **同步内容** - 选择同步的设置项
- **立即同步** - 手动触发同步

### 本地备份

- **自动备份** - 定期自动备份设置
- **备份位置** - 备份文件存储位置
- **恢复备份** - 从备份恢复设置

## 快捷操作

| 操作 | 快捷键 |
|------|--------|
| 打开设置 | `Ctrl+,` |
| 切换深色模式 | `Ctrl+Shift+D` |
| 切换夜间模式 | `N` |
| 重置当前页设置 | `Ctrl+Shift+R` |
