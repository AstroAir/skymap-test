# 快速开始

欢迎来到 SkyMap Test 文档！本章节将帮助您快速上手使用 SkyMap Test 天文观测应用。

## 本章内容

1. **[安装指南](installation.md)** - 详细的安装说明，支持 Windows、macOS 和 Linux
2. **[首次运行](first-run.md)** - 首次启动时的配置向导
3. **[基本概念](basic-concepts.md)** - 天文学基础知识，包括坐标系统和时间系统
4. **[功能导览](quick-tour.md)** - 主要功能的快速介绍

## 系统要求

### Windows

- Windows 10 或更高版本（64位）
- 至少 4GB 内存
- 500MB 可用磁盘空间

### macOS

- macOS 10.15 (Catalina) 或更高版本
- Intel 或 Apple Silicon 处理器
- 至少 4GB 内存
- 500MB 可用磁盘空间

### Linux

- 主流 Linux 发行版（Ubuntu 20.04+, Fedora 33+, 等）
- 至少 4GB 内存
- 500MB 可用磁盘空间

## 快速上手

如果您是第一次使用 SkyMap Test，建议按照以下顺序阅读：

### 1. 安装应用

根据您的操作系统，参考[安装指南](installation.md)完成应用安装。

### 2. 首次配置

首次启动时，应用会引导您完成基本配置：

- 设置观测位置
- 选择星图 survey
- 配置显示选项

### 3. 了解基本概念

阅读[基本概念](basic-concepts.md)了解：

- 天球坐标系
- 时间系统
- 天体坐标表示

### 4. 探索功能

通过[功能导览](quick-tour.md)快速了解主要功能：

- 星图导航
- 天体搜索
- 观测规划
- 设备管理
