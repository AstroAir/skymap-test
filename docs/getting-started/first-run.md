# 首次运行配置

本文档将指导您完成首次运行 SkyMap Test 时的初始配置。

## 配置向导

首次启动 SkyMap Test 时，应用会自动启动配置向导，引导您完成基本设置。

### 步骤 1：设置观测位置

观测位置是计算天体位置和可见性的基础。

#### 位置设置方式

1. **从列表选择**
   - 从预设城市列表中选择最近的城市
   - 列表包含全球主要城市

2. **手动输入坐标**
   - 纬度：-90° 到 +90°
   - 经度：-180° 到 +180°
   - 海拔高度（可选）

3. **自动定位**（未来功能）
   - 使用 GPS 或网络定位服务

#### 时区设置

- 应用会自动检测系统时区
- 您也可以手动选择时区
- 时区对于准确的天文时间计算很重要

### 步骤 2：配置显示选项

#### 星图 Survey 选择

选择默认的星图 survey：

- **HIPPARCOS** - 依巴谷星表，精度高，恒星数量适中
- **Tycho-2** - 第谷星表，恒星数量更多
- **UCAC4** - 美国海军天文台星表，位置精度极高

#### 星等限制

设置显示的最暗星等：

- **默认值**：6.0 等（肉眼可见极限）
- **推荐值**：
  - 城市观测：4.0-5.0 等
  - 郊区观测：5.5-6.5 等
  - 山区观测：7.0 等以上

#### 星座连线

选择是否显示星座连线和边界：

- **星座连线**：连接恒星形成星座图案
- **星座边界**：显示星座的官方边界
- **星座名称**：显示星座名称

### 步骤 3：下载基础数据

首次运行需要下载基础数据：

#### 下载数据内容

- **星表数据**：约 100MB
- **天体名称**：约 10MB
- **星座边界数据**：约 5MB

#### 下载时间

- 宽带连接：约 1-5 分钟
- 可以稍后在设置中下载

#### 下载位置

数据会下载到：

- **Windows**: `C:\Users\<用户名>\AppData\Local\SkyMapTest\cache\`
- **macOS**: `~/Library/Application Support/SkyMapTest/cache/`
- **Linux**: `~/.local/share/SkyMapTest/cache/`

## 基本界面介绍

### 主界面布局

```mermaid
graph TD
    A[主窗口] --> B[顶部菜单栏]
    A --> C[左侧工具栏]
    A --> D[中央星图显示区]
    A --> E[右侧信息面板]
    A --> F[底部状态栏]

    B --> B1[文件]
    B --> B2[编辑]
    B --> B3[视图]
    B --> B4[工具]
    B --> B5[帮助]

    C --> C1[导航控制]
    C --> C2[缩放控制]
    C --> C3[时间控制]
    C --> C4[图层控制]

    D --> D1[星图渲染]
    D --> D2[天体标记]
    D --> D3[坐标网格]

    E --> E1[天体信息]
    E --> E2[可见性信息]
    E --> E3[目标列表]

    F --> F1[当前时间]
    F --> F2[观测位置]
    F --> F3[坐标信息]
```

### 主要功能区域

#### 1. 星图显示区

- 中央区域，占据大部分窗口
- 显示实时星空
- 支持鼠标交互操作

#### 2. 工具栏

- 左侧垂直排列
- 包含常用工具快捷按钮

#### 3. 信息面板

- 右侧显示
- 显示选中天体的详细信息
- 可通过按钮显示/隐藏

#### 4. 状态栏

- 底部显示
- 显示当前时间、位置、坐标等信息

## 基本操作

### 鼠标操作

- **左键拖动**：旋转视角
- **滚轮**：缩放
- **右键拖动**：平移
- **双击**：居中显示

### 键盘快捷键

- **方向键**：旋转视角
- **+/-**：缩放
- **T**：跳转到当前时间
- **Space**：暂停/恢复时间

### 基本工作流程

1. **设置观测位置和时间**
2. **浏览星图**，熟悉界面
3. **使用搜索功能**定位熟悉的天体
4. **调整显示选项**，获得最佳视觉效果
5. **保存配置**，作为默认设置

## 配置文件位置

配置文件保存在：

- **Windows**: `C:\Users\<用户名>\AppData\Roaming\SkyMapTest\`
- **macOS**: `~/Library/Application Support/SkyMapTest/`
- **Linux**: `~/.config/SkyMapTest/`
